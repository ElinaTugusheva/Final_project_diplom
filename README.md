# Final Project Diplom - Система автоматизации закупок

## Описание проекта
Backend-приложение для автоматизации закупок в розничной сети через REST API.

Сервис предоставляет возможность заказа товаров из нескольких магазинов. Пользователь собирает заказ (корзину), добавляя товары разных магазинов, представленные в едином каталоге. После подтверждения заказа, он сохраняется в базе со статусом "Новый".

## Технологии
- **Python** 3.10+
- **Django** 4.2.7
- **Django REST Framework** 3.14.0
- **JWT Authentication** (djangorestframework-simplejwt)
- **PostgreSQL** / SQLite (для разработки)
- **Celery** (для асинхронных задач)
- **Redis** (брокер сообщений)
- **Docker** (контейнеризация)

## Функциональность

### Базовая часть:
- Регистрация и авторизация пользователей
- Подтверждение email
- Каталог товаров с фильтрацией и поиском
- Корзина покупок
- Оформление заказов
- Управление контактами (адресами доставки)
- Импорт товаров от поставщиков (YAML)
- Отправка уведомлений по email

### Продвинутая часть (опционально):
- Экспорт товаров
- Админка заказов (проставление статуса и уведомление клиента)
- Асинхронные задачи (Celery)

## Основные сущности

1. **Пользователь** - покупатель или поставщик
2. **Магазин** - поставщик товаров
3. **Категория** - категория товаров
4. **Товар** - единица товара
5. **Информация о товаре** - цена, количество, характеристики в конкретном магазине
6. **Корзина** - временный заказ пользователя
7. **Заказ** - подтвержденный заказ
8. **Контакт** - адрес доставки пользователя
